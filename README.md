### express-api-starter

Quick start

- Install dependencies:

```
npm install
```

- Copy the example environment file and update values as needed:

```
cp sample.env .env
```

- Run the project in development mode:

```
npm run dev
```

If you need to reinstall all packages later, run `npm install` again.
